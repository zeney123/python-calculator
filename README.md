# python-calculator
A simple command-line calculator built in Python.  This project supports basic arithmetic operations like addition, subtraction, multiplication, and division.
# Features
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers
- ASCII art logo
- Continue calculation with previous result
